Creating An NPM Project

1. In the command windown go to the tech directory
2. Create a new directory
c:\tech> mkdir practice
3. Navigate to the new directory
c:\tech> cd practice
4. Create a new npm project with default values
5. Open the directory with VS Code
c:\tech\practice> code .
6. Open the terminal in VS Code
c:\tech\practice> code .
7. Install http-server
8. Update NPM Scripts To Run The http-server
9. Start The Server

Creating An HTML Page

1. In VS Code create a new file called index.html
2. Add the following code into the index.html file
            
                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Practice Title</title>
                </head>
                <body>
                    
                </body>
                </html>
3. Add a message to between the body tag.
4. View your changes in the browser.

Creating and link a CSS file

1. In VS Code create a new file called index.css
2. Create a CSS Selector for body with a rule for the color red.
3. Link the CSS File.
4. View The Changes.

Creating and load a JS file

1. In VS Code create a new file called index.js
2. Write a line of code that alerts a message to the window.
3. Load the script.
4. Reload the window.
